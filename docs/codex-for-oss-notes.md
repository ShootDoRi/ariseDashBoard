# Codex For OSS Notes

## Repository Qualification

Community Ops Dashboard is an open-source dashboard template for community maintainers who manage rankings, members, and activity through spreadsheets. It is based on a live community operations dashboard already used in production, and is being generalized with sample data, CI, docs, and maintainer workflows.

## API Credits Use

We will use API credits with Codex for PR review, issue triage, release notes, security review, and generating or maintaining spreadsheet adapters so community maintainers can support new schemas without manual coding.

## Evidence To Attach Privately

- Anonymous operating scale, such as number of maintainers, active members, and weekly views.
- Before-and-after workflow notes showing time saved versus manual spreadsheet review.
- Redacted screenshots or screen recordings that show the dashboard in use without exposing private member data.
- Release and maintenance activity after the public `v0.1.0` tag.
