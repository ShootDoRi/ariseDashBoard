# Roadmap

## v0.1

- Publish the generalized dashboard with anonymous sample data.
- Support config-driven Google Sheets sync.
- Add CI for tests, build, and secret scanning.

## v0.2

- Add schema presets for common community ranking sheets.
- Add import validation warnings in the dashboard UI.
- Add screenshot/GIF demo assets.

## v0.3

- Add adapter generation workflow for new spreadsheet schemas.
- Add role-based dashboard sections for maintainers, moderators, and contributors.
- Add optional release-note and issue-triage automation.
